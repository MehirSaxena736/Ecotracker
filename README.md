---

## Overview

EcoCheck is a desktop application developed using Java Swing that helps users calculate, monitor, and analyze their daily and monthly carbon emissions. The application offers interactive dashboards, graphical analytics, carbon budget tracking, PDF report generation, and personalized sustainability recommendations to encourage environmentally responsible decisions.

Users can calculate emissions generated from transportation, electricity consumption, and lifestyle activities while monitoring their progress toward monthly carbon reduction goals.
